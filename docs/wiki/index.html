01 · Movement & Collision

Set up a player, walls, and a basic map loader.

// Player.hx (skeleton)
class Player extends FlxSprite {
  public function new(x:Float,y:Float){
    super(x,y);
    makeGraphic(10,10,0xFFFFFFFF);
  }
  override public function update(elapsed:Float){
    velocity.set(0,0);
    if(FlxG.keys.pressed.LEFT)  velocity.x = -60;
    if(FlxG.keys.pressed.RIGHT) velocity.x =  60;
    if(FlxG.keys.pressed.UP)    velocity.y = -60;
    if(FlxG.keys.pressed.DOWN)  velocity.y =  60;
    super.update(elapsed);
  }
}
02 · Dialogue & Typewriter

typewriter with skip and auto‑advance.

class Typewriter {
  public static function write(t:FlxText, s:String, cps=30){
    var i=0; FlxTimer.start(1/cps, function(timer){
      t.text += s.charAt(i++);
      if(i>=s.length) timer.cancel();
    }, 0);
  }
}
