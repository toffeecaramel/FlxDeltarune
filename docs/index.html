Downloading the framework

As you (maybe) probably know, the framework has TWO builds that can be downloaded.

The first one is the PLAYER build, which is the build players will use to play mods.

The second one is the DEV build, which is the one you're looking for, made for developing mods.

So the first thing you want to do is to download it from this link

Accessing the Modding Tools

This is probably the easiest part! All you gotta do is press CTRL + SHIFT + M on the chapter select screen to open the modding tools menu.

If you did everything correctly, you should be able to see this:

Scripting cheats

Importing everything on the engine

Below is a piece you can paste on the beginning of a script to import all of the engine's classes.

import backend.effects.*;
import backend.game.*;
import backend.game.interfaces.*;
import backend.scripting.events.*;
import backend.utils.*;
import backend.*;
import frontend.mods.*;
import game.battle.*;
import game.chars.*;
import game.editors.*;
import game.ui.*;
